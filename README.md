# Gestion de projets – LE STUDIO GARMENT

## Infos projet

Application de gestion de projets pour la production textile, construite avec React, TypeScript, Vite, shadcn-ui et Supabase.

## Comment lancer le projet en local ?

Vous pouvez éditer l’application avec l’IDE de votre choix.

Pré‑requis : Node.js & npm installés – par exemple via [nvm](https://github.com/nvm-sh/nvm#installing-and-updating).

Étapes :

```sh
# Étape 1 : Cloner le dépôt avec l’URL Git de votre projet.
git clone <VOTRE_GIT_URL>

# Étape 2 : Aller dans le dossier du projet.
cd <NOM_DU_PROJET>

# Étape 3 : Installer les dépendances.
npm install

# Étape 4 : Lancer le serveur de dev (hot reload).
npm run dev
```

## Technologies utilisées

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Déploiement

Vous pouvez déployer ce projet sur la plateforme de votre choix (par ex. Netlify, Vercel, Render) en pointant vers la commande de build `npm run build` et le dossier `dist`.
